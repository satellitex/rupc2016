Problem A: Gossip

Problem

n人の人が横一列に並んでいる。 各人には1からnまでの番号が順番に割り当てられている。

人iは一単位時間で人i-1と人i+1に情報を伝達することができる。
ただし、i=1の人はi=2の人にしか情報を伝達できず、i=nの人はi=n-1の人にしか情報を伝達できない。

0秒時点でm人のa1, a2, ..., amの人が秘密の情報を手に入れた。
すべての人が秘密の情報を得ることのできる最小の時間を求めよ。

Input

入力は以下の形式で与えられる。

n m
a1 a2 ... am

1行目に2つの整数nとmが空白区切りで与えられる。2行目にm個の整数a1, a2, ..., amが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ n ≤ 105
  • 1 ≤ m ≤ n
  • 1 ≤ ai ≤ n
  • aiの値は全て異なる。
  • aiは昇順で与えられる。

Output

すべての番号の人に情報が伝わる最小時間を1行に出力する。

Sample Input 1

3 2
1 3

Sample Output 1

1

Sample Input 2

10 3
2 5 7

Sample Output 2

3

Sample Input 3

10 5
2 5 6 8 10

Sample Output 3

1

Sample Input 4

100000 1
1

Sample Output 4

99999
