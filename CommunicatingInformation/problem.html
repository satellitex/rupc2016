Problem A: Message

Problem

番号1~nのn人の人が横1列に並んでいる。 ある番号iの人は、番号i-1とi+1の人に情報を合わせて1秒で伝達することができる。 0秒時点でm人の番号a1, a2, ..., amの人が情報を知っている。すべての番号の人に情報が伝わる最小の時間を求めよ。

Input

入力は以下の形式で与えられる。

n m
a1 a2 ... am

1行目に2つの整数nとmが空白区切りで与えられる。2行目にm個の整数a1, a2, ..., amが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ n ≤ 105
  • 1 ≤ m ≤ n
  • 1 ≤ ai ≤ n
  • aiの値は全て異なる。
  • aiは昇順で与えられる。

Output

すべての番号の人に情報が伝わる最小時間

Sample Input 1

3 2
1 3

Sample Output 1

1

Sample Input 2

10 3
2 5 7

Sample Output 2

3

Sample Input 3

10 5
2 5 6 8 10

Sample Output 3

1

Sample Input 4

100000 1
1

Sample Output 4

99999
