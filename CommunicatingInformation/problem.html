Problem A: Gossip

12プロこと会津プロダクションは有名芸能プロダクションである。12プロには多くのアイドルが在籍している。 この問題セット（の前半）はそんな12プロのアイドルたちのシンデレラストーリーである……

Problem

アイドルがn人横一列に並んでいる。 各アイドルには1からnまでの番号が順番に割り当てられている。

アイドルiは単位時間でアイドルi-1とアイドルi+1に噂を伝達することができる。
ただし、アイドル1はアイドル2にしか噂を伝達できず、アイドルnはアイドルn-1のアイドルにしか噂を伝達できない。

0秒時点で番号がa1, a2, ..., amであるm人のアイドルがプロデューサーについての噂を手に入れた。
すべてのアイドルがプロデューサーの噂を得ることのできる最小の時間を求めよ。

Input

入力は以下の形式で与えられる。

n m
a1 a2 ... am

1行目に2つの整数nとmが空白区切りで与えられる。
2行目にm個の整数a1, a2, ..., amが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 2 ≤ n ≤ 105
  • 1 ≤ m ≤ n
  • 1 ≤ ai ≤ n
  • aiの値は全て異なる。
  • aiは昇順で与えられる。

Output

すべての番号のアイドルにプロデューサーの噂が伝わる最小時間を1行に出力する。

Sample Input 1

3 2
1 3

Sample Output 1

1

Sample Input 2

10 3
2 5 7

Sample Output 2

3

Sample Input 3

10 5
2 5 6 8 10

Sample Output 3

1

Sample Input 4

100000 1
1

Sample Output 4

99999
