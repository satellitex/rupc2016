Problem H: Max coin

Problem

Cocoさんは、銅貨をx枚、銀貨をy枚持っている。

n人の硬貨交換人がいる。 それぞれの交換人iは、

  • Cocoさんの銅貨aiと交換人の銀貨bi枚
  • Cocoさんの銀貨ciと交換人の金貨di枚
のどちらか1つの方法で1回だけ交換してくれる。

この制約の下で、最終的に所持している金貨の枚数を最大化せよ。

Input

入力は以下の形式で与えられる。

n
x y
a1 b1 c1 d1
a2 b2 c2 d2
...
an bn cn dn

1行目に整数nが与えられる。2行目に整数x, yが空白区切りで与えられる。3行目からn+2行目に整数ai, bi, ci, diが空白区切りで与えられる。

Constraints

入力は以下の条件を満たす。

  • 1 ≤ n ≤ 100
  • 0 ≤ x, y ≤ 300
  • 1 ≤ ai, bi, ci, di ≤ 300
  • ai, bi, ci, diの合計はそれぞれ300を超えない。

Output

最終的に所持している金貨の枚数の最大値を1行に出力する

Sample Input 1

3
3 3
3 2 4 1
5 1 2 1
3 1 3 2

Sample Output 1

3

Sample Input 2

3
3 4
3 1 3 1
3 1 4 1
3 1 2 1

Sample Output 2

2

Sample Input 3

4
5 0
5 1 1 1
2 1 2 1
4 1 1 1
3 1 3 1

Sample Output 3

2

Sample Input 4

4
0 0
1 10 1 10
2 5 2 5
3 25 5 6
1 15 2 20

Sample Output 4

0
